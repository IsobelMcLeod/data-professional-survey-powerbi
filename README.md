# data-professional-survey-powerbi
Interactive Power BI dashboard analysing salary trends, job roles, satisfaction, and programming language preferences among data professionals.
## Key Insights
- Average salary by job title (Data Scientist, Data Engineer, Data Analyst, etc.)
- Geographic distribution of survey respondents
- Most popular programming languages among data professionals
- Perceived difficulty of breaking into the data industry
- Levels of satisfaction with work/life balance and salary

## Tools & Technologies
- Power BI
- Power Query (data cleaning and transformation)
- Data visualisation
- DAX (basic measures)
